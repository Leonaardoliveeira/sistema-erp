Integração via Script (API)

Rota usada pelo Backup Agent para registrar backups automaticamente:

POST /api/backup
Authorization: Bearer <token>

{ "clienteId": "<ObjectID>", "status": "ok", "tamanho": "2.3 GB",
  "destino": "MEGA", "dataBackup": "2024-01-15T03:00:00" }

Verificar bloqueio antes de executar: GET /api/clientes/:id/status-backup
Retorna {"bloqueado": false} se liberado para backup.
