Integração via Script (API)

Chame esta rota no seu script de backup para registrar automaticamente após o upload para a nuvem:

POST /api/backup
Authorization: Bearer <seu_token>
Content-Type: application/json

{
  "clienteId":  "<ObjectID do cliente>",
  "status":     "ok",
  "tamanho":    "2.3 GB",
  "destino":    "MEGA/Backups/gestores",
  "observacao": "Banco: gestores | Servidor: localhost",
  "dataBackup": "2024-01-15T03:00:00"
}

O clienteId é o ObjectID obtido na tabela acima.
Verifique se o backup está suspenso antes de executar: GET /api/clientes/:id/status-backup
