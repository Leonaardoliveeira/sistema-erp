Integração via Script (API)

Chame esta rota no seu script de backup para registrar automaticamente após o upload para a nuvem:

POST /api/backup
Authorization: Bearer <seu_token>
Content-Type: application/json

{
  "clienteId": "<id_do_cliente>",
  "status":    "ok",
  "tamanho":   "2.3 GB",
  "destino":   "S3",
  "dataBackup":"2024-01-15T03:00:00"
}

Para obter seu token, faça login via POST /api/login e use o token retornado.
O campo status aceita: "ok", "falha" ou "pendente".
